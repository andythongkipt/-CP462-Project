## 📌 Project Overview

This project focuses on analyzing and predicting website vulnerability levels using machine learning techniques.  

The objective is to identify whether a website is considered **High Risk** or **Low Risk** based on selected security-related features.

The project includes:

- Data collection and preprocessing
- Data cleaning and normalization
- Exploratory Data Analysis (EDA)
- Feature scaling using MinMaxScaler
- Logistic Regression modeling
- Model evaluation and interpretation

---

## 📊 Methodology

### 1️⃣ Data Wrangling

- Statistical analysis using SciPy

### 3️⃣ Model Development

## 📈 Results

The model successfully classified website vulnerability into two categories:

- High Risk
- Low Risk

The evaluation metrics demonstrate the effectiveness of logistic regression for binary classification in cybersecurity risk prediction.
